## 关于学习的摘抄
> You don’t need to learn everything, you just need to be curious about learning. When the time comes that you need it, you’ll be prepared.

## 便于记忆的集中方法
```
- 把长时间的学习分割为几个单位，使“最后”和“最先”的情况大大增加，理想的学习时段是10分钟到45分钟，中间休息时间2-10分钟。
-  运用连锁关系和出色醒目。即编造本不相干的A、B关系，使其练习起来。然后把A、B夸大，使其变得荒唐或者幽默，而且尽可能想象自己触碰到它的外形，听见它的声音，看见它的画面，闻到它的气味等等。

- 每天，去观察人的某个特定部位，多多观察其特异性，就比较容易记住自己所遇见的新面孔。

- 学习一个时段后休息大约10分钟，然后首次复习，隔一天再进行第二次复习，1周以后第三次复习，1个月以后第四次，4个月后第五次复习。

- 用关键字句，特殊的画图记日记。

- 有意识地反复回忆一些事情。

- 经常在脑中展现生动形象，色彩丰富的画面，这样能训练右脑的思维。
```

## 训练听力
```
- 坐下来放松全身，然后用两手的中指分别抵住左右耳屏(外耳门前面的突起)，使之堵住耳门，把注意力集中在耳部两三分钟或更久一些，

- 直到你听到隐隐有雷声传来，再突然松开双手，然后闭幕静听周围或远处一些细碎的声音。

- 聆听时分心通常有两大因素：一是环境，一是我们自己内在的思绪。脑子可以关掉听力，也可以打开听力。不妨选择某个地方，分开听各种不同的声音。

- 不要注意对方流水般的句子，设法注意对方话语里的关键词句。

- 尽力运用你所能运用的一切感觉聆听这个世界的每一种声音，无论它是一群暴徒出现的声音，还是谦如君子柔若淑女的声音。这样，你或许将感到你所驻足的这个不停转动的星球正沸腾着无数的生命。

- 外围视力的准确是因为视网膜的1亿5000万光线接收体中，只有一小部分（不及5%）在敏锐视力区，其他都属外围视力。为了增强这种能力，请将视力集中在前方，

- 练习由眼角看东西。眼睛看着前方并盯着一个物体，维持焦点不动，四下游览房间（用“肉眼”）来看。
```

## 保养眼睛
```
- 人在高度紧张的时，会忘了眨眼睛，这样一来，眼睛会过度疲劳，无法润滑，眼睛的血液也不流通。所以你专心做一件事情的时候，务必要经常眨眼经，并不时把视线转开一会，让眼睛休息。

- 眼睛工作一段时间后，闭上眼睛，用双手遮住光线，手不要碰到眼睛，只在上面搭成圆顶状，此时不妨想象世界一片漆黑。

- 人专心的时候，呼吸会受阻，致使身体各部分紧张，视力也连带受害。所以你长时间专注于某件事的时候，呼吸务必要轻松有规律，这对视力将大有帮助。

- 利用视觉导杆可使阅读速度提高一倍，并改进领悟力，理解力和记忆力。导杆集中了注意焦点，鼓励眼睛顺畅均匀地活动，防止眼光倒退和游移的坏毛病。

- 使用导杆最有效的方法是：指着你正在阅读的字眼的下方，保持移动性，使眼睛能挑出恰当的字群。
```

## 研究性阅读的4个过程
```
- 综览（浏览并抓住资料的要点）

- 预习（回忆浏览的内容并思考）

- 详读（深入挖掘问题）

- 复习（选择性的回顾）
```

## 记笔记
```
- 关键字的研读：在合理的范围内，关键字越少越好。关键字该在最简短的情况下表达最贴切的意思，而做笔记的人受测时可以最快回想出来。

- 事实上我们听见，说出和看见的字句中，只有1%到10%是必要的关键字眼。当我们想告诉某人一个特殊的事情或情况时，并不是一些拖拉冗长的句子排着队等

- 着你把他们说出来。你脑子里只有重要物体的关键字眼和画面，你依此构想出表达概念所需要的句子。

- 当我们对一件事物完全体会时，文字的意义已显得不重要。

- 创造性笔记：利用箭头、颜色和特殊的符号，你可以连接不同的关键字，使人立刻看出其中的关系。具体的方法有：标不同的颜色、列大纲、图案化、

- 方案大小变化、底下划线等。

- 实际加快速度：

    - 缩写，遇到繁复的词与单字，可以用自己熟知的记号来替代。

    - 定期做“1分钟快写”的训练。一般人每份钟可写25到35个具有10画左右的字。长期坚持...
```

## 创造力
```
- 有意识地去感受和记忆你所经历的一切事物，并不是没有意义的事情。

- 你所做的，你所经验的————如果你把更有趣的事情与你的言谈联系起来，对此多加思索，那你就有更多的实力和吸引力。

- 当沃恩的行为被某种情感，信念和生命附着的时候，它们就会具备感染力。
```

## 感染力
```
- 一种坚定的你认为正确的信仰；

- 一种坚定不移的生命哲学；

- 一种与众不同的情感表达方式。
```

## 讨论
```
另一个极其重要的做法就是与人讨论，因为讨论不可避免涉及到两个环节：

表达自己的思维：这就是前面说的，把自己的思维表达出来，思维本身就变成了你思考的对象，你就有可能注意到思维环节当中之前没有注意到的隐蔽的环节，并对其“取反”，跳出原本的框框。

参照别人的思维：由于每个人思维的框框（隐含假设）未必一样，所以你的“显然”可能是别人的“不一定吧？”。通过对比，之前隐形的思维边界就会显现出来，这就好像在黑夜中没法看到黑色的字，而在白色背景的映衬下就会一目了然。
```
